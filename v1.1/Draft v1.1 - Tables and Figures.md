## 使用说明

### 如何将表格插入到对应markdown文件

1. **表1**：复制到"Draft v1.1 - Sec tions 1-3.md"的第3.4节末尾
2. **表2**：复制到"Draft v1.1 - Sections 5-8.md"的第5.2节位置
3. **表3**：复制到"Draft v1.1 - Sections 5-8.md"的第6.1节位置
4. **表4**：复制到"Draft v1.1 - Sections 5-8.md"的第6.3节位置
5. **表5和表6**：复制到"Draft v1.1 - Sections 5-8.md"的第7节相应位置

### 如何处理图片

图片文件位于`outputs/figures/`目录，在markdown中使用以下格式引用（路径需根据实际位置调整）：

正文引用示例：
```
如图1所示，诊断性强度参数在样本期内...

![图1：诊断性强度参数β_t的时变路径](outputs/figures/beta_t.png)
```

### 格式一致性检查清单

- [ ] 所有表格使用三线表格式（仅顶线、中线、底线）
- [ ] 表格标题在表格上方
- [ ] 表注（*注*:）在表格下方
- [ ] 图片说明在图片下方
- [ ] 所有表格和图在正文中有引用
- [ ] 表号、图号连续编号
- [ ] 数值精度一致（通常保留2-3位小数）
- [ ] 变量名称中英文一致

### 交叉引用检查

在正文中引用表格和图时，使用以下格式：
- "表1报告了..."
- "如图1所示..."
- "从表2可以看出..."

确保所有提到的表号、图号都实际存在。
