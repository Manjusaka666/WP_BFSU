## 整合步骤

### 方法1：手动整合（推荐）

1. 创建新文件 `Draft v1.1 - Complete.md`

2. 按顺序复制粘贴以下文件内容：
   ```
   Draft v1.1.md
   Draft v1.1 - Section 4.1.md（接在第3节后）
   Draft v1.1 - Section 4.2-4.3.md（接在4.1节后）
   Draft v1.1 - Sections 5-8.md（接在第4节后）
   Draft v1.1 - References and Appendices A-C.md（接在第8节后）
   Draft v1.1 - Mathematical Appendices D-F.md（接在附录C后）
   ```

3. 检查章节编号连贯性（应从1-8，附录A-F）

4. 检查公式编号（建议按章节重新编号，如4.1, 4.2等）

### 方法2：使用命令行（Windows）

在项目根目录打开命令行，执行：

```cmd
type "Draft v1.1 - Sections 1-3.md" "Draft v1.1 - Section 4.1.md" "Draft v1.1 - Section 4.2-4.3.md" "Draft v1.1 - Sections 5-8.md" "Draft v1.1 - References and Appendices A-C.md" "Draft v1.1 - Mathematical Appendices D-F.md" > "Draft v1.1 - Complete.md"
```
